# Tradedesk

Library and infrastructure code for running automated trading strategies with IG

## Required ENV vars

```bash
# IG API Credentials
# Get these from your IG account at https://www.ig.com/uk/myig/settings/api-keys
IG_API_KEY=your_api_key_here
IG_USERNAME=your_username
IG_PASSWORD=your_password

# Environment: DEMO or LIVE
# Use DEMO for testing with demo account, LIVE for real trading
IG_ENVIRONMENT=DEMO

# Logging level: DEBUG, INFO, WARNING, ERROR
LOG_LEVEL=INFO
```
